Display: flex

In this assignment, most of the contents were used flexbox for layout arrangement.

Commands used included:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content

During this assignment I got to test and learn different values such as flex-start/end, spac between/around in order to justify the content on the page.

for example the navigation bar is arrange using flex.

A sample code for this:

.navigator{

background-color: rgba(43, 43, 43, 0.8);

width: 100%;

display: flex;

flex-flow: row wrap;

justify-content: space-around;

margin: auto;

overflow: hidden;

padding: 0px;

}
